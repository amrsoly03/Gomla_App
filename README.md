# 🛒 Gomla Express App

Gomla Express is a modern Flutter-based e-commerce application designed to simplify wholesale shopping. The app connects users with suppliers, allowing them to browse products, manage orders, and access exclusive offers, all within a clean and scalable architecture.

# 🚀 Features
🛍 Browse products across multiple categories

🔍 Smart search & filtering to quickly find products

❤️ Favorites system to save preferred items

🛒 Cart management with real-time updates

📦 Order placement & tracking for a smooth purchasing experience

🔥 Exclusive offers & discounts for better deals

🏢 Company & category browsing for organized shopping

⚡ Pagination support for optimized performance with large datasets

📦 Clean architecture (MVVM) with scalable and maintainable codebase

# 🛠 Tech Stack

Flutter

Dart

Cubit (flutter_bloc)

RESTful APIs

Clean Architecture (MVVM)

Pagination & Infinite Scrolling

Local Caching & State Handling
